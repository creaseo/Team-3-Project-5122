# North Carolina Fiber Opportunity Explorer

## Project Description
Interactive view of county-level broadband need, work-from-home demand, income, and fiber revenue potential across North Carolina. This tool helps identify the best county targets for fiber-optic expansion by combining broadband need, remote-work demand, and income strength into a unified Opportunity Score.
